pinboard.in opera extension

An extension for Opera to quickly bookmark pages to pinboard.in

Features

  • Click the toolbar button to get a pre-filled submit form with the title and url of the current page
  • A list of your tags for quickly adding/removing tags before bookmarking
  • A list of suggested tags for the current page
  • Edit the details of the bookmark if the current page was saved before
  • "Read later" button for quickly saving the page as to-read
